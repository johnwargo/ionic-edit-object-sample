Ionic Edit Object Sample The standard mechanism for Ionic apps is to use the automatic, reactive UI. Any changes made to an Object are automatically reflected in the app UI. Tap Edit (No Cancel) below to see an example of this. Edit (No Cancel) What many apps need, however, is a way to display an edit dialog, but not accept any changes until the user taps the Save or OK button. When they tap Cancel, the object reverts to its original state. Tap Edit (With Cancel) below to see an example of this.
